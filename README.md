[![Telegram Bot](https://img.shields.io/badge/Telegram-Bot-2CA5E0?logo=telegram)](https://t.me/currency32converterbot)

A powerful and user-friendly Telegram bot for real-time currency conversion.

## ✨ Features

- 🚀 Instant conversion in any chat
- 🎯 Interactive conversion with buttons
- 💬 Quick conversion via text: `100 USD to EUR`
- 📊 Live exchange rates
- 🌍 30+ currencies supported
- 📱 Works in groups and private chats
- 🔄 Multiple API fallbacks for reliability

## 📋 Commands

| Command | Description |
|---------|-------------|
| `/start` | Start the bot and see welcome message |
| `/help` | Show all available commands |
| `/convert` | Start interactive conversion |
| `/rate USD EUR` | Check exchange rate between two currencies |
| `/currencies` | List all supported currencies |
| `/about` | About this bot |
