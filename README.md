# 💱 Currency32 Converter Bot

A Telegram bot for real-time currency conversion.

## Features
- Convert between 20+ currencies
- Interactive conversion with buttons
- Quick conversion via text (e.g., `100 USD to EUR`)
- Check exchange rates
- Support for major world currencies

## Commands
- `/start` - Start the bot
- `/help` - Show help
- `/convert` - Interactive conversion
- `/currencies` - List supported currencies
- `/rate` - Check exchange rate
- `/about` - About the bot

## Quick Usage
Just type: `100 USD to EUR` in any chat!

## Deployment
Deployed on Railway using GitHub.

## Tech Stack
- Python 3.11
- python-telegram-bot
- ExchangeRate-API
- Railway
